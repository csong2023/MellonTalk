## Getting Started

To execute the project, follow these instructions:

### Installation


1. Clone the repo
   ```sh
   git clone https://github.com/csong2023/MellonTalk.git
   ```
2. Install Python modules
   ```sh
   python3 -m pip install SpeechRecognition
   brew install portaudio
   pip install pyaudio
   ```
3. Run the code!
   ```sh
   Command + B / Ctrl + B
   ```
